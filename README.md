# 📦 insport AI Chatbot - 구축 및 실행 가이드

대장님, **'antigravity_chatbot_guide'**를 기반으로 구축한 **insport 전용 AI 챗봇 시뮬레이션**입니다.

## 🚀 빠른 실행 방법
1.  **워크스페이스 이동:** `cd 'd:\안티그래비티\chatbot for insport\insport-chatbot'`
2.  **종속성 설치:** `npm install` (이미 설치 전이라면)
3.  **로컬 서버 실행:** `npm run dev`

## 🛠 주요 구현 기능
-   **카카오톡 테마 UI:** 실제 오픈빌더와 유사한 버튼 메뉴 및 대화 환경.
-   **Claude Vision 시뮬레이션:** 사진 전송 시 `defect`, `severity` 등을 판별하는 AI 로직 모사.
-   **Slack 연동 (Sim):** 관리자용 알림 대시보드 및 n8n 워크플로우 상태 확인.
-   **프리미엄 디자인:** Glassmorphism, Neon Blue Accent, 미려한 애니메이션.

## 📁 프로젝트 구조
-   `src/App.jsx`: 메인 챗봇 로직 (React).
-   `src/index.css`: 프리미엄 디자인 시스템 (Vanilla CSS).
-   `index.html`: SEO 최적화 및 캐시 방지 적용.
